# Vinculación y publicación en GitHub Pages

En este apartado se describe cómo se vincula el repositorio creado con GitHub Pages para mostrar la documentación web generada por MkDocs.

## Pasos realizados

1. Activación de GitHub Pages desde las opciones del repositorio (Settings > Pages).
2. Selección de la rama `gh-pages` como fuente y configuración de la carpeta raíz.
3. Obtención de la URL de la web pública en GitHub Pages.
4. Verificación de que la documentación está accesible desde la dirección proporcionada.
